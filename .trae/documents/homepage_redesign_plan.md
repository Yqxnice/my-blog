# 博客首页重新设计方案

## 项目概述
重新设计博客首页，移除"我的博客"标题，改为左侧大文字特效区域，右侧大头像区域，并集成联系方式。

## 当前状态分析
**现有首页结构**：
- 页面标题："我的博客"
- 时间进度条
- 最新文章 + 热门标签
- 全部文章网格

**需要移除**：
- "我的博客"标题区域

**需要添加**：
- 左侧：打字机特效文字
- 右侧：大头像
- 联系方式（微信等）

## 设计方案

### 布局结构
```

│  │               │        │                │       │
```

### 详细设计

#### 1. 头部区域（新）

**左侧：文字特效区域**
- 主标题："木子博客" - 大字体，特效动画
- 副标题1："技术分享" - 中等字体
- 副标题2："生活感悟" - 中等字体
- 特效：文字渐入、颜色变化、轻微缩放

**右侧：头像和联系方式**
- 大头像：圆形，带边框和阴影
- 联系方式：
  - 微信二维码（可悬停显示）
  - 邮箱地址
  - GitHub 链接
  - Twitter 链接
- 社交图标：悬停效果

#### 2. 时间进度条（保留）
- 保持现有功能
- 样式优化

#### 3. 内容区域（保留并优化）
- 最新文章展示
- 热门标签云
- 全部文章网格
- 响应式布局优化

## 技术实现细节

### 文字特效
- 使用 CSS 动画
- 文字渐变效果
- 逐字出现动画
- 悬停时微动画

### 头像区域
- 圆形头像：`border-radius: 50%`
- 头像边框：`border: 4px solid #f3f4f6`
- 头像阴影：`shadow-xl`
- 响应式大小：
  - 桌面：200px × 200px
  - 平板：150px × 150px
  - 移动：120px × 120px

### 联系方式
- 社交图标：Lucide React
- 悬停效果：颜色变化 + 缩放
- 微信二维码：悬停显示
- 链接处理：新窗口打开

### 响应式设计
- 桌面（> 1024px）：左右布局
- 平板（768px - 1024px）：左右布局，缩小尺寸
- 移动（< 768px）：上下布局，头像在上

## 实施步骤

### 步骤 1：移除现有标题区域
- 移除 `h1` 标题 "我的博客"

### 步骤 2：创建新的头部布局
- 添加左右分栏容器
- 实现左侧文字特效区域
- 实现右侧头像和联系方式

### 步骤 3：添加文字特效
- 实现标题动画

### 步骤 4：实现头像和联系方式
- 添加头像图片

- 调整不同断点的布局

- 测试动画性能

## 预期效果

1. **视觉冲击**：大头像 + 特效文字，第一印象强烈
2. **个人品牌**：突出博主形象和联系方式
3. **现代感**：动画效果提升页面活力
4. **专业度**：清晰的布局和交互
5. **用户体验**：快速找到联系方式，易于导航

## 风险评估

- **性能风险**：动画可能影响性能 → 优化动画，使用 `will-change`
- **响应式风险**：小屏幕布局可能拥挤 → 设计移动优先
- **兼容性风险**：某些动画可能在旧浏览器不支持 → 优雅降级
