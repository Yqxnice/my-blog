# 时间进度条组件 - 实现计划

## 项目概述
在博客首页添加一个时间进度展示区域，显示今天、本月、本年的已过时间进度，包含实时更新的进度条。

## 实现步骤

### [ ] 1. 创建时间进度条组件
- **位置**: `components/blog/TimeProgress.tsx`
- **功能**: 
  - 计算今天已过时间（时、分、秒）
  - 计算本月已过时间（天、小时）
  - 计算本年已过时间（月、天）
  - 每秒实时更新今天进度
  - 显示对应的进度条和百分比
- **设计要点**:
  - 使用现代简约风格
  - 支持深色模式
  - 响应式布局
  - 平滑的进度条动画

### [ ] 2. 集成到首页
- **位置**: 首页个人简介下方或最新文章上方
- **方式**: 在 `app/(blogs)/page.tsx` 中导入并使用 TimeProgress 组件
- **布局**: 可以作为一个独立区域，或者与统计信息结合

### [ ] 3. 样式优化
- 确保与现有设计风格一致
- 进度条颜色搭配主题色
- 添加适当的过渡动画
- 确保在移动端的显示效果

### [ ] 4. 测试与验证
- 测试组件在不同时间段的显示
- 验证今天进度的实时更新
- 检查深色模式下的显示效果
- 测试响应式布局

## 技术细节

### 时间计算逻辑
- **今天进度**: 
  - 已过秒数 / 一天总秒数 (86400)
  - 每秒更新
- **本月进度**:
  - 已过天数 / 本月总天数
  - 每分钟更新
- **本年进度**:
  - 已过月数 / 12
  - 每小时更新

### 组件状态管理
- 使用 `useState` 存储当前时间
- 使用 `useEffect` 设置定时器每秒更新
- 组件卸载时清理定时器

### 进度条设计
- 使用 Tailwind CSS 的宽度百分比
- 添加过渡动画实现平滑效果
- 显示具体数值和百分比

## 验收标准

- [ ] 本年进度正确显示已过月数
- [ ] 进度条动画流畅
- [ ] 支持深色模式
- [ ] 响应式设计，移动端友好
- [ ] 与首页整体风格协调
